# magick tool

### Instal magick tool SO

- archlinux

```
sudo pacman -Sy
sudo pacman -S imagemagick
```

- ubuntu

```
sudo apt update
sudo apt install imagemagick
```

### Example magick convert

- archlinux

```
magick file.webp output-file.jpg
```

- ubuntu

```
convert file.webp output-file.jpg
```
